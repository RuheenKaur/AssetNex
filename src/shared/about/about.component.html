About AssetNexIT

Built to solve a real problem

AssetNexIT is a full-stack IT asset management system developed as part of a structured software engineering training programme - designed to reflect real-world enterprise practices from day one.

What is AssetNexIT?

AssetNexIT is a centralized platform for managing IT assets across an organization. It eliminates manual tracking and replaces spreadsheets and emails with a structured, role-based digital system.

Admins have full control over assets, users, assignments, and support tickets. Users can view their assigned equipment and raise support requests - all from a clean, responsive interface.

Architecture at a Glance

Every layer built with industry-standard practices.

01

Angular Frontend

Standalone components, reactive forms, HTTP interceptors for JWT, route guards for role protection, and paginated tables with real-time search and sort.

02

.NET 8 Web API

Layered architecture - Controllers, Services, Repositories. Async/await throughout, global exception handling, API versioning, and Swagger with Bearer auth.

03

Azure Cloud

App Service hosts the API. Blob Storage serves Angular. Key Vault stores all secrets. Managed Identity handles authentication — no credentials in code anywhere.

04

CI/CD Pipeline

GitHub Actions across two repos. Angular builds and uploads artifacts. API pipeline downloads and deploys to Azure — fully automated on every push to main.

Built on solid principles

Every decision was made with reasoning - not just implementation.

Security First

JWT tokens, role-based authorization, Key Vault secret management, and passwordless SQL via Managed Identity.

SOLID Architecture

Repository pattern, dependency injection, interface segregation - each layer has a single responsibility.

Performance Aware

AsNoTracking on read queries, paginated APIs, debounced search, and lazy loading on Angular routes.
